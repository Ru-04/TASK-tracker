# Task Tracker Web App

A simple and modern Task Tracker web application built using HTML, CSS, and JavaScript.  
This app helps users add tasks, track pending work, mark tasks as completed, and visualize progress using a circular progress indicator.

---

##  Features

-  Add new tasks
-  View all pending tasks
-  Mark tasks as completed
-  Automatically move completed tasks to the completed section
-  Circular progress bar showing task completion percentage
